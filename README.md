# CO2削減サポート

建材見積書をAIが読み取り、環境配慮型製品（CR LIMEX 等）への置き換えによる **コスト差額** と **CO₂削減効果**、さらに **GX-ETS排出枠換算額** を自動算出する、建築企業向けのSaaSです。

- **マルチテナント対応** — 会社ごとにアカウント・データを完全分離
- **どんな形式の見積書でも対応**（PDF / Excel / CSV / 画像 / テキスト）
- **自社製品データベース**を画面上で管理（単価・CO₂係数・照合キーワード・算定根拠）
- **GX-ETS排出枠換算** — 削減CO₂を2026年度の政府公表価格で円換算
- **セキュリティ** — bcryptパスワード認証・レート制限・監査ログ・権限管理
- **完全無料構成で運用可能**（Render Free + Neon Free、いずれも期限切れなし）

---

## 主な機能

### 見積診断
見積書をアップロードすると、AIが明細を読み取り、自社製品への置き換え候補を照合。項目ごとにコスト差額・CO₂削減率を算定し、営業提案用のコメントと経営層向け総括を自動生成します。第三者検証済みの製品には「検証済」バッジが付きます。

### 製品データベース（会社ごと）
自社の環境配慮型製品を登録。単価・CO₂排出係数・置換対象の基準CO₂・照合キーワードに加え、**CO₂算定根拠**（LCA算定方法・検証機関）と**第三者検証フラグ**を保持でき、診断結果の信頼性を担保します。

### ユーザー・権限管理
- **オーナー** … 会社の最初の登録者。全操作＋ユーザー管理が可能
- **管理者** … ユーザー管理・監査ログ閲覧が可能
- **メンバー** … 見積診断・製品編集が可能

### 監査ログ
ログイン、製品編集、ユーザー追加、見積診断などの操作を記録。オーナー・管理者が設定画面で確認できます。

## アーキテクチャ

3層構造で「AIの創造性」と「数値の正確性」を分離しています。

1. **抽出（AI）** — 見積書をAnthropic APIが読み取り、明細を構造化
2. **照合・計算（コード）** — 製品DBと照合し、コスト・CO₂・GX-ETS換算額を**確定的に**計算（数値はAIに委ねない）
3. **提案文生成（AI）** — 確定数値をもとに提案コメントを生成（API失敗時も定型文で結果を必ず表示）

データ（会社・ユーザー・製品・設定・ロゴ・監査ログ）はすべてPostgreSQLに保存。アプリ本体はステートレスで、永続ディスクを必要としません。

## デプロイ手順（完全無料・期限切れなし構成）

### 1. Neonで無料PostgreSQLを作成
1. [neon.tech](https://neon.tech) で無料アカウントを作成（クレジットカード不要・期限切れなし）
2. プロジェクトを作成（リージョンはSingapore等、Renderに近い場所を推奨）
3. **Connection String** をコピー（`postgres://...neon.tech/...?sslmode=require`）

### 2. Renderにデプロイ
1. このリポジトリをGitHubにpush（`render.yaml` はリポジトリのルート直下に置く）
2. Render → **New** → **Blueprint** → リポジトリを選択
3. 環境変数を設定：
   - `DATABASE_URL` … NeonのConnection String
   - `ANTHROPIC_API_KEY` … Anthropic APIキー
   - `SESSION_SECRET` … 自動生成される（そのままでOK）
4. **Deploy Blueprint** をクリック

初回アクセス時にDBスキーマが自動作成されます。`/healthz` でヘルスチェック可能です。

### 3. 最初の会社を登録
デプロイ後、トップページの「新規登録」から会社名・担当者・メール・パスワードを入力すると、オーナーアカウントとサンプル製品が作成されます。

## セキュリティ

- **パスワード認証** — bcrypt（コスト12）でハッシュ化。平文は保存しません
- **セッション** — PostgreSQLに保存されるHTTP-onlyクッキー。本番ではSecure属性・8時間有効
- **レート制限** — ログイン/登録（15分に10回）、見積診断（1分に12回）、一般API（1分200回）
- **マルチテナント分離** — 全クエリが org_id でスコープされ、他社データにアクセス不可
- **権限制御（RBAC）** — ユーザー管理・監査ログはオーナー/管理者のみ
- **監査ログ** — 主要操作を記録
- **セキュリティヘッダー** — Helmet（CSP・HSTS・frameguard等）
- **アップロード** — すべてメモリ処理（ディスク非書き込み）＋サイズ制限
- APIキーはサーバー環境変数として保持され、ブラウザには一切露出しません

## 環境変数

| 変数 | 必須 | 説明 |
|------|:----:|------|
| `DATABASE_URL` | ✓ | PostgreSQL接続文字列 |
| `ANTHROPIC_API_KEY` | ✓ | 見積診断のAI解析に使用 |
| `SESSION_SECRET` | ✓ | セッション署名用（Renderで自動生成） |
| `NODE_ENV` | | `production` でセキュアCookie・HSTS有効化 |
| `ANTHROPIC_MODEL` | | 使用モデル（既定 `claude-sonnet-5`） |
| `ALLOW_SIGNUP` | | `false` で新規会社登録を無効化 |
| `MAX_FILE_MB` / `MAX_LOGO_MB` | | アップロード上限 |
| `ETS_PRICE_LOW` / `ETS_PRICE_HIGH` | | GX-ETS参考価格（円/t-CO₂） |

## ローカル開発

```bash
npm install
cp .env.example .env      # DATABASE_URL, ANTHROPIC_API_KEY 等を設定
npm start                 # http://localhost:3000
```

ローカルにPostgreSQLが必要です（`localhost` 接続時はSSLを自動的に無効化します）。

## 無料構成の制約（把握しておくべき点）

- **Render無料Webサービス** は15分間アクセスがないとスリープし、次回アクセス時に30〜60秒の再起動が発生します（期限切れ・データ削除はありません）。
- **Neon無料PostgreSQL** は期限切れなし・0.5GB・月100 CU時間。本アプリの用途では十分ですが、アクセス集中が続く場合はNeon有料プランへ（コード変更不要）。

本格運用でスリープを避けたい場合は、RenderのWebサービスを有料プラン（月$7〜）にアップグレードしてください。

## 既知の注意点

- `xlsx`（SheetJS）のnpm版には脆弱性勧告があります。本番では公式CDN版への差し替えを推奨します:
  ```bash
  npm rm xlsx && npm i https://cdn.sheetjs.com/xlsx-latest/xlsx-latest.tgz
  ```
- 単価・CO₂係数は製品DBの登録値に基づく試算です。対外提案には第三者検証済みのデータをご登録ください（「検証済」バッジで区別表示されます）。
